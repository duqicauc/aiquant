# 模型优化思路 - 2025-12-28

本文档记录模型优化的思路和计划。

---

## 一、当前模型优化思路

### 技术面因子

1. **增加换手率（自由流通股）**
   - 换手率是衡量股票流动性的重要指标
   - 使用自由流通股计算更准确

2. **增加乖离率**
   - 前复权、长中短期乖离率
   - 反映价格偏离均线的程度

3. **增加EMA（指数移动平均）**
   - 相比简单移动平均，EMA对近期价格更敏感
   - 可以捕捉趋势变化

4. **增加KDJ指标**
   - 随机指标，反映超买超卖状态
   - 适合短期交易信号

5. **增加TOPDAYS**
   - 连续上涨/下跌天数
   - 反映趋势持续性

6. **增加OBV（能量潮）**
   - 成交量指标，反映资金流向
   - 量价关系分析

7. **成交量与5日、20日平均成交量的比值**
   - 量比指标
   - 反映当前成交量与历史平均的对比
