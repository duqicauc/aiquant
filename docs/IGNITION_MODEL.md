# 点火期模型开发文档 🔥

## 模型概述

点火期模型旨在识别**启动初期**的潜力股，核心目标是：
- **在股票已启动但未大涨时识别**（涨幅10-30%区间）
- **捕捉突破信号，提前布局**
- **在右侧初期买入，降低时间成本**

## 核心思路

### 设计理念

与现有模型不同，点火期模型关注的是**启动初期的特征**，而非底部或已起爆的特征。

**现有模型对比**：
- **三连阳模型**：识别已起爆的强势股（涨幅>50%），属于右侧追涨
- **左侧起爆点模型**：识别底部潜力股（涨幅<20%），属于左侧布局
- **点火期模型**：识别启动初期（涨幅10-30%），属于右侧初期

### 正样本定义

对于每只股票，找到满足以下条件的T1时点：

1. **启动初期特征（T1前34天）**：
   - 累计涨幅在 **10% - 30%** 之间（已启动但未大涨）
   - 价格处于启动初期阶段，有上涨动能但涨幅未过大

2. **突破信号（T1前7天）**：
   - 量比 > 2.0（成交量明显放大）
   - MACD金叉或即将金叉
   - 其他技术指标突破信号

3. **验证条件（T1后3周）**：
   - 3周内累计涨幅 > 50%（验证确实成为牛股）
   - 确保不是假突破

4. **排除条件**：
   - ST股票、上市不足180天、北交所股票、停牌/退市股票

### 模型优势

1. **时机更优**：在启动初期识别，比三连阳模型更早，比左侧模型更确定
2. **风险可控**：已有10-30%涨幅证明启动，但未大涨避免追高风险
3. **信号明确**：突破信号（量比>2、MACD金叉）提供明确买入时机
4. **符合实战**：更贴近实际投资场景，在股票启动初期买入

### 特征工程重点

- **启动幅度特征**：T1前34天涨幅、涨幅趋势、价格位置
- **突破信号特征**：T1前7天量比、MACD状态、均线突破
- **技术指标特征**：RSI、KDJ、布林带等指标状态
- **成交量特征**：量比变化、成交量趋势、放量持续性

### 与现有模型的关系

三个模型形成完整的选股体系：
- **左侧起爆点模型**：底部识别，提前布局（左侧）
- **点火期模型**：启动初期识别，右侧初期买入（右侧初期）
- **三连阳模型**：已起爆识别，右侧追涨（右侧）
